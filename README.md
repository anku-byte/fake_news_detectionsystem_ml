# 📰 Fake News Detection System with WhatsApp Integration 🤖

## 📌 Project Overview

This project is an AI-powered Fake News Detection System that classifies news as **REAL** or **FAKE** using Machine Learning. It also verifies news using live data from News APIs and allows users to check news directly via WhatsApp.

---

## 🚀 Features

* ✅ Fake News Detection using Machine Learning (TF-IDF + Random Forest)
* 🌐 Web Interface using Flask
* 📡 News Verification using API
* 📱 WhatsApp Bot Integration using Twilio
* ⚡ Real-time prediction with confidence score

---

## 🧠 Technologies Used

* Python
* Machine Learning (Scikit-learn)
* Natural Language Processing (TF-IDF)
* Flask (Web Framework)
* Twilio API (WhatsApp Integration)
* News API
* HTML, CSS

---

## ⚙️ How It Works

1. User inputs news text (Web / WhatsApp)
2. Text is processed using TF-IDF Vectorizer
3. Machine Learning model predicts **REAL** or **FAKE**
4. News API verifies from trusted sources
5. Result is displayed to user
